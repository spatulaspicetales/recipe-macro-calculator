# 🍳 Recipe Macro Calculator

> A free, lightweight nutrition calculator built for home cooks and food creators. No sign-up, no ads, no fluff — just instant macro breakdowns for any recipe.

**[➜ Try it live](https://github.com/spatulaspicetales/recipe-macro-calculator/)** ← replace with your URL

---

## What it does

- 🔍 Search 60+ common foods with instant autocomplete
- ✏️ Add custom ingredients with your own nutrition values
- 🍽️ Set servings and get per-serving macros automatically
- 📊 See calorie split across protein, carbs, and fat
- 📧 Built-in email signup for weekly recipes

## Why I built this

I'm a software engineer and food creator — and I got tired of apps that required accounts just to calculate macros for a simple meal. This is a single HTML file. No tracking, no login, no nonsense.

## How to use

1. Type an ingredient name in the search box
2. Set the gram amount
3. Click **Add**
4. Repeat for all ingredients
5. Set your servings — macros update instantly

## Food database

60+ foods sourced from USDA FoodData Central including:
- Proteins: chicken, beef, fish, eggs, tofu, legumes
- Carbs: rice, pasta, bread, oats, potatoes, fruits
- Fats: avocado, oils, nuts, dairy
- Vegetables: broccoli, spinach, peppers, tomatoes

Want a food added? Open an issue or submit a pull request.
